# Decision Structures (if / elif / else)

## Bu derste öğrendiklerim
- if / elif / else
- and / or operatörleri
- Karşılaştırma operatörleri (< , >, <=, >=, ==)
- Negatif giriş doğrulama (Input Validation)
- Sınır değer kontrolü (Boundary Checking)
- Gerçek hayata uygun veri seçimi

## Mini projeler
- Gym Membership – Yaşa göre üyelik kontrolü
- Driver License – Ehliyet alma uygunluğu
- Cinema Ticket – Yaşa göre bilet türü
- Cash Withdrawal – ATM para çekme işlemi
- Market Discount System – Alışveriş indirimi hesaplama
- Student Grading System – Nota göre harf notu belirleme
- User Login – Kullanıcı adı ve şifre doğrulama

## Öğrendiğim Yazılım Prensipleri
- Anlamlı değişken isimleri kullanmak
- Kullanıcı hatalarını kontrol etmek
- Sınır değerleri test etmek
- Gerçek hayat problemlerini koda dönüştürmek
